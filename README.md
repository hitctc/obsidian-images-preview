# Images Preview (Obsidian Plugin)

Click images in your notes to open popup previews with zoom, drag, transform, gallery navigation, and Pin mode.

## Features

- Normal mode: single-image popup with mask, toolbar, and optional gallery bar.
- Pin mode: multi-image floating previews with max count and cover strategy.
- Operations: zoom in/out, 1:1, fit, copy, close.
- Mobile gestures: single-finger drag, two-finger pinch, double tap to toggle fit/1:1, long-press menu.
- Strong lifecycle: uses `registerEvent` / `registerDomEvent`, avoids duplicate listeners.
